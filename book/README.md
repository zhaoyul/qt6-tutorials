# 跨语言 Qt6 开发指南

> C++ · Python · Clojure · Basilisp

---

## 本书简介

本书是一本全面的 Qt6 跨语言开发指南，涵盖了四种不同编程语言（C++、Python、Clojure、Basilisp）使用 Qt6 进行应用程序开发的实践方法。

### 为什么选择 Qt6？

Qt6 是一个跨平台的 C++ 应用程序开发框架，提供了：
- **丰富的模块**：GUI、网络、数据库、多媒体等
- **跨平台能力**：一次编写，随处运行（Windows、macOS、Linux、移动设备、嵌入式）
- **现代 UI**：支持传统的 Widgets 和现代的 QML/Quick
- **优秀的性能**：原生编译，响应迅速
- **活跃的社区**：丰富的文档和示例

### 为什么选择多语言？

不同的项目需求适合不同的编程语言：

| 语言 | 适用场景 | 特点 |
|------|----------|------|
| **C++** | 高性能应用、系统级开发 | 原生性能、完整 Qt 功能 |
| **Python** | 快速原型、数据科学、脚本 | 简洁语法、丰富生态 |
| **Clojure** | 函数式编程、并发处理 | Lisp 语法、不可变数据 |
| **Basilisp** | Python 生态 + Lisp 优雅 | Python 互操作、现代 Lisp |

### 学习方法建议

1. **循序渐进**：建议按照章节顺序学习，从基础核心开始
2. **动手实践**：每个示例都配有完整代码，务必亲自运行和修改
3. **对比学习**：可以同时查看多种语言的实现，加深理解
4. **项目驱动**：完成最后的综合项目，巩固所学知识
