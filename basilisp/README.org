* PySide6 Qt6 教程 - Basilisp 版本

本文件夹提供了与 Python 示例相对应的 Basilisp 重写版本.

** 环境要求

- Python 3.9+
- PySide6 (=pip install pyside6=)
- Basilisp (=pip install basilisp=)

** 运行示例

从仓库根目录运行:

#+begin_src bash
PYTHONPYCACHEPREFIX=/tmp/pycache conda run -n qt6-py python -m basilisp.cli run basilisp/01_core/01_meta_object/main.lpy
#+end_src

或从示例目录运行:

#+begin_src bash
cd basilisp/01_core/01_meta_object
PYTHONPYCACHEPREFIX=/tmp/pycache conda run -n qt6-py python -m basilisp.cli run main.lpy
#+end_src

注意事项:
- Basilisp 示例直接基于 PySide6 编写(不使用 =runpy=)
- QML/Quick 演示从 =python/= 目录加载 QML 文件, 因此请从仓库根目录运行以保持相对路径一致

** 目录结构

目录结构与 =python/= 一一对应:

#+begin_example

#+end_example
